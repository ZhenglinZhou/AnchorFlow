AnchorFlow
Training-Free 3D Editing via Latent Anchor-Aligned Flows
Paper
ArXiv
Code

AnchorFlow is a training-free framework for 3D shape editing. It aligns source and target flow trajectories with latent anchors to achieve semantic-consistent, identity-preserving, mask-free 3D editing. (Click the teaser below to see AnchorFlow in action 👇)
